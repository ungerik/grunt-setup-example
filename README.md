NextRealityEngine
=================

A WebGL 3D Engine

Tech Stack
----------

* [Node.js](http://nodejs.org/)
* [grunt](http://gruntjs.com/)
* [component](https://github.com/component/component)
* [bower](http://twitter.github.com/bower/)

Installation
------------

	$ component install ungerik/NextRealityEngine
